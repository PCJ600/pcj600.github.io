趋势科技(中国)有限公司Trend Vision One™ Service Gateway2022.4-至今开发人员

简介：

Service Gateway是一个安装在企业客户本地网络的服务网关,允许本地的趋势产品访问云服务, 共享威胁情报,减少客户流量.
服务网关由两部分组成: 一个本地的Linux虚拟设备, 和一个云端的APP, 用于管理连接的虚拟设备.
技术：Linux系统升级, kubernetes, MQ, Python, REST API, Shell

贡献：

  • 设计了一种Linux双系统分区的在线升级方案, 支持虚拟设备从CentOS一键迁移到Rocky, 解决了CentOS停止维护导致的安全漏洞问题, 和升级故障后无法回滚的问题, 显著改善了用户体验.
  • 优化了微服务的升级方案. 用Python实现了一个后台服务, 连接AWS IoT并接收云端APP的请求, 实现微服务的在线安装和升级, 从而把微服务从固件中解耦. 基础镜像从4.5G压缩到2G, 客户下载时间节约了50%.
  • 使用Microk8s Ingress提供统一的入口点, 简化微服务的管理和集成.
  • 基于CLISH开发了一个CLI, 允许客户配置虚拟设备. 用Shell实现了一组配置命令, 实现注册, 网络配置等功能；设计了一组诊断命令, 用于定位OS,k8s,网络问题, 解决客户本地网络中调试不便的困难.

一个安装在Service Gateway上的微服务, 为本地的趋势终端产品接入云服务提供正向代理功能,减少客户网络带宽消耗.
技术：Squid, HTTP, Python

贡献：

华为技术有限公司Matrix Simulation Platform2019.8-2022.4开发人员

简介：

Matrix是一个基于Docker和Redis实现的交换机仿真平台, 用于设备管理领域的开发人员在没有物理设备的情况下进行代码测试, 提高开发者测试效率.
技术：Docker, Redis, C

贡献：
